# MedDiagnostic

Веб-приложение медицинской диагностической компании, разработанное с
использованием фреймворка Django.

------------------------------------------------------------------------

## Описание проекта

Система предназначена для: - предоставления информации о компании -
отображения медицинских услуг - регистрации и авторизации
пользователей - записи на приём - работы с обращениями клиентов -
администрирования через панель администратора

------------------------------------------------------------------------

## Функциональность

### Публичная часть

-   Главная страница
-   О компании
-   Услуги
-   Контакты

### Пользовательская часть

-   Регистрация и авторизация
-   Личный кабинет
-   Запись на приём
-   Просмотр записей

### Админка

-   Управление услугами
-   Управление врачами
-   Работа с записями
-   Обращения пользователей

------------------------------------------------------------------------

## Технологии

-   Python
-   Django
-   PostgreSQL
-   Bootstrap 5
-   Docker

------------------------------------------------------------------------

## Запуск через Docker

``` bash
docker compose up --build
docker compose exec web python manage.py migrate
docker compose exec web python manage.py createsuperuser
```

------------------------------------------------------------------------

## Тестирование

``` bash
docker compose exec web python manage.py test
```
